# Update
```
yarn set version stable
```

# Start a new project
```
yarn init
```

# Installing all the dependencies
```
yarn
yarn install
```

# Adding a dependency
```
yarn add [package]
yarn add [package]@[version]
yarn add [package]@[tag]
```

# Removing a dependency
```
yarn remove [package]
```
